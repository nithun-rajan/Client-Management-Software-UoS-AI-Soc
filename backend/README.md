# Estate Agent CRM - Backend API

FastAPI backend for the Estate Agent Client Management System.

## Quick Start

```bash
# Install dependencies
uv sync

# Run server
uvicorn app.main:app --reload
```

## Features

- Property Management
- Landlord Management
- Applicant Tracking
- Advanced Search
- KPI Dashboard
- Event Logging
- **DevEx & QA Infrastructure** (CI/CD, observability, security)

## Documentation

- API Docs: http://localhost:8000/docs
- Health Check: http://localhost:8000/health
- Metrics: http://localhost:8000/metrics
